Zapatec.Busycal

Zapatec.Busycal()
    Zapatec Busy Calendar widget class. Extends Zapatec Calendar class. 
     Accepts JSON source only.

 Input data format:
 {
   busydates: [
     array of busy dates as Date objects or strings in RFC 2445 (yyyymmdd) or
     ISO 8601 (yyyy-mm-dd) format
   ]
 }

 If callbackSource config option is defined, callback function receives range
 of displayed dates:
 {
   from: [string] first displayed date in RFC 2445 format,
   to: [string] last displayed date in RFC 2445 format
 }

 In addition to config options defined in base Zapatec.Calendar class
 provides following config options:

 preloadMonths [number] Number of months to preload. 0 - no preload.
 Default: 1.
    Parameters:
      any - Any number of arguments passed to parent method
    Requires:
      - Requires zpcal/src/calendar.js
